# FLUSSO 8 — Operativo Web / Tech / Automazioni

**Trigger:** "sito", "seo", "analytics", "automazioni", "email marketing", "funnel"

## SEQUENZA AGENTI

```
Tecnico Web + Ottimizzatore + Misuratore (KPI) → Voce (copy se necessario)
```

## FASI

### Fase 1 — Analisi tecnica
- **Agente:** Tecnico Web
- **Task:** Audit tecnico (SEO, performance, analytics setup, automazioni esistenti)
- **Output:** /clients/[cliente]/progetti/[nome]/findings/tecnico_audit.md

### Fase 2 — Ottimizzazione
- **Agente:** Ottimizzatore
- **Task:** CRO, funnel optimization, UX improvements, automazioni
- **Output:** /clients/[cliente]/progetti/[nome]/findings/ottimizzatore_raccomandazioni.md

### Fase 3 — KPI e tracking
- **Agente:** Misuratore
- **Task:** Setup KPI, dashboard, tracking plan
- **Output:** /clients/[cliente]/progetti/[nome]/findings/misuratore_tracking.md

### Fase 4 — Copy (condizionale)
- **Agente:** Voce
- **Condizione:** Solo se servono testi per landing page, email sequence, funnel copy
- **Task:** Copy ottimizzato per conversione
- **Output:** /clients/[cliente]/progetti/[nome]/findings/voce_copy.md

## NOTE
- Stratega solo se serve una decisione strategica (es: cambio di piattaforma, rebranding)
- Per CRO e funnel, l'Ottimizzatore guida il flusso
- Tecnico Web e Ottimizzatore possono lavorare in parallelo se i task sono indipendenti
