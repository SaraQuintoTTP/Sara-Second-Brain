# FLOW 7 — Training / Academy

**Triggers:** "course", "training", "workshop", "academy", "teaching material"

## AGENT SEQUENCE

```
Trainer (structure + content) + Voice (copywriting) → Narrator (presentations)
```

## PHASES

### Phase 1 — Structure & Content
- **Agent:** Trainer
- **Task:** Design course structure, learning objectives, modules, exercises
- **Output:** `/clients/[client]/projects/training_[name]/findings/trainer_structure.md`

### Phase 2 — Content Copywriting
- **Agent:** Voice
- **Task:** Write module text content (persuasive copy, storytelling)
- **Output:** `/clients/[client]/projects/training_[name]/findings/voice_content.md`

### Phase 3 — Presentations
- **Agent:** Narrator
- **Task:** Create pptx presentations for modules
- **Output:** `/clients/[client]/projects/training_[name]/findings/narrator_slides.pptx`

## NOTES
- Add Mentor if individual coaching is needed within the training program
- Add Strategist if the course requires strategic framing (e.g., training for a client's marketing team)
- Content must be practical and actionable — no pure theory
