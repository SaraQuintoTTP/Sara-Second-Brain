## Skill da valutare per evoluzioni future

### impeccable — Design Language per Frontend UI
**Data valutazione:** 2026-05-19
**Fonte:** [github.com/pbakaus/impeccable](https://github.com/pbakaus/impeccable)

**Cos'è:** Skill Claude Code che aggiunge una design language al workflow frontend. 1 skill / 23 comandi (`/audit`, `/polish`, `/typeset`, `/critique`…), 7 reference file (typography, color OKLCH, motion, spatial, interaction, responsive, UX writing), 27 anti-pattern deterministici. Elimina il look "AI-generated" dalle UI in codice (React/HTML/CSS).

**Perché non ora:** TTP è un'agenzia marketing/comunicazione — i deliverable core sono piani, copy, deck, social. Non costruisce frontend per i clienti. Le 17 design skill già presenti nel Global Skills Arsenal (es. `ui-ux-pro-max`, `frontend-design`, `canvas-design`) sono ancora inassegnate.

**Quando rivalutare:** Se TTP aggiunge servizi di web design o landing page build come offerta dedicata. In quel caso, assegnare a Optimizer (CRO) o a un futuro agente "Web Designer", dopo aver prima attivato `ui-ux-pro-max` e `frontend-design`.
